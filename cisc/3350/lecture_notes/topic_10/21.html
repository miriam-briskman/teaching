Printing a Text Description (String) of a Signal

Name: 
psignal()
 System call or function: Function Links: Online Manual | Course Packet
What it does: directly prints to stderr the signal description of 
signo
, preceded by the message 
msg
that you indicate.
What libraries you must include: 
#include <signal.h> // Defines psignal().
Syntax: 
void psignal (int signo, const char *msg);
Description of arguments: 
signo
: a signal code such as SIGINT, SIGTERM, etc.

msg
: a char array containing the message you want to print before the description is printed.
What type it returns: 
void
 On success: – This function doesn't return anything –
On failure: – This function doesn't fail –
If failure, does it set 
errno
? No
Quick example: 
psignal (SIGHUP, "Here is the description");
// Will print the following to stderr:
// Here is the description: Hangup
Other variations: 
strsignal()
