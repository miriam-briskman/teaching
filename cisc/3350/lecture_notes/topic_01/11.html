API vs. ABI

The source code of prog1.c.

The binary code of prog1.c, represented as a string of 0s and 1s.

The source code on the left will compile on ANY Linux machine because the functions and code used in it belong to the Linux API. However, the binary code on the right will run only on some Linux machines. Every device type has its own ABI, and only if the ABIs of both machines are the same, you can copy over the executable/binary file from one device to another and successfully run it on both!
