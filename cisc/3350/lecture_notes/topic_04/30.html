Positional Writes

Name: 
pwrite()
 System call or function: System call Links: Online Manual | Course Packet
What it does: writes data from an array into a file without changing the file position, and returns the # of written bytes.
What libraries you must include: 
#define _XOPEN_SOURCE 500 // extra function definitions + pread().
#include <unistd.h> // Defines pwrite(), size_t, ssize_t, and off_t.
Syntax: 
ssize_t pwrite (int fd, void *buf, size_t len, off_t pos);
Description of arguments: 
fd
: file descriptor of an open file

buf
: The array name (or address of a variable, e.g., &var) from which data is taken.

len
: Maximum number of bytes we want to write.

pos
: The position (byte) in the file from where you want to start writing.
What type it returns: 
ssize_t
 On success: A non-negative integer indicating how many bytes we wrote. This number will be equal to or smaller than 
len
, and could also be 
0
(in cases of some special files.)
On failure: 
-1
If failure, does it set 
errno
? Yes
Quick example: 
ssize_t number_of_chars_written = pwrite (fd, arr, strlen(arr), 50);
Other variations: N/A
