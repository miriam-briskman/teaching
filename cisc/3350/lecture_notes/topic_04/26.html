Truncating the File

If the file "hello.txt" contains the following 2 lines of text, what will happen if we run the program below?

#include <unistd.h>
#include <stdio.h> 
int main()
{
    int ret = truncate ("./hello.txt", 45);
    if (ret == 1)
    {
        perror ("truncate");
        return 1;
    }
    return 0;
}
