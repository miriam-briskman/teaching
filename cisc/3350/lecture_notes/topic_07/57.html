Session System Calls

Name: 
setsid()
 System call or function: System Call Links: Online Manual | Course Packet
What it does: creates a new session (only if the calling process isn't some group's leader,) and a group inside this session, and then becomes both the session's and the group's leader; both the 
sid
and 
pgid
are set to the process's 
pid
.
What libraries you must include: 
#include <unistd.h> // Defines setsid().
Syntax: 
pid_t setsid (void);
Description of arguments: – This system call doesn't accept any argument. –
What type it returns: 
pid_t
 On success: The ID of the created session.
On failure: 
-1
If failure, does it set 
errno
? Yes
Quick example: 
pid_t new_session = setsid();
if (new_session == -1) /* Handle error */
Other variations: N/A

The only possible error 

errno
value is 
EPERM
, indicating that the process is already the leader of an existing process group.
