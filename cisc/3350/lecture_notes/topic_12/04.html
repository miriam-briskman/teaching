Getting/Reading a File's Metadata

Name: 
fstat()
 System call or function: System Call Links: Online Manual | Course Packet
What it does: reads the metadata of the file with file descriptor 
fd
and pastes it into 
buf
(an instance of the 
stat
structure.)
What libraries you must include: 
#include <sys/stat.h> // Defines fstat().
Syntax: 
int fstat (int fd, struct stat *buf);
Description of arguments: 
fd
: the file descriptor (fd) of a file that was open with 
open()
.

buf
: a 
stat
 structure that will afterwards contain the file's metadata.
What type it returns: 
int
 On success: 
0
On failure: 
-1
If failure, does it set 
errno
? Yes
Quick example: 
struct stat file_status;
int fd = open("/cisc3350​/assignments​/text.txt", O_RDWR);
if (fstat (fd, &file_status) == -1)
    perror("fstat");
Other variations: 
stat()
, 
lstat()
