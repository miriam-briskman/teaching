Finding What the Current Working Directory (cwd) Is

Name: 
getcwd()
 System call or function: Function Links: Online Manual | Course Packet
What it does: Pastes the absolute path to the current working directory of this running process into 
buf
, which is of length 
size
bytes. That is, after 
getcwd()
returns, 
buf
will contain the current working directory.
What libraries you must include: 
#include <unistd.h> // Defines getcwd().
Syntax: 
char * getcwd (char *buf, size_t size);
Description of arguments: 
buf
: a char array where the path to the directory will be copied to.

size
: the size of 
buf
 in bytes.
What type it returns: 
char*
 On success: A non-
NULL
pointer to 
buf
.
On failure: 
NULL
If failure, does it set 
errno
? Yes
Quick example: 
char buf[1000]; // buffer on the stack
if (!getcwd (buf, 1000))
   /* Handle the errors */
printf ("The current working directory is: %s\n", buf);
Other variations: N/A
