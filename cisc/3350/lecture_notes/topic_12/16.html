Example of Changing a File's Group

// getgrnam() returns information on a
// group, given the group's name:
struct group *gr = getgrnam ("cats");
if (!gr) // Probably an invalid group.
{
    perror ("getgrnam");
    exit (EXIT_FAILURE);
}

// Set info.txt's group to 'cats':
if (chown("info.txt", -1, gr->gr_gid))
    perror ("chown");

The code on the left changes the group of the info.txt file to cats. The 

group
data structure is defined as follows:

struct group {
    char  *gr_name;   /* group name */
    char  *gr_passwd; /* group password */
    gid_t gr_gid;     /* group ID */
    char  **gr_mem;   /* NULL-terminated pointers array
                         to names of group members */
};

After running this code, a call to 

$ ls -l
will show that the file's group name indeed changed to cats.
