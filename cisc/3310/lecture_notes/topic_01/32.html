Practical Example: Which Desktop to Buy?

Now that we understood what these terms mean, we can go back to answer our question.
Since the most critical hardware component for games to run without lagging or crashing is a high RAM size, Johnny should get a device with greater RAM size (nowadays, at least 8 GB.) Also, having more computer cores/threads will improve the responsiveness of the game, so Johnny should strive towards more cores/threads, if available. Since Desktop №3 has the greatest RAM size, and since the number of cores is also large, Johnny should choose Desktop №3 out of the given 3 desktops.
