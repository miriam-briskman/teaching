Looking Up Info Online about a Computer

Fun class activity: This page lists the details of an HP computer model. Where on this page can you find:

  1. The number of CPU cores in this model's CPU?
  2. The size in Megabytes or Gigabytes of main memory?
  3. The size in Gigabytes or Terabytes of secondary storage
  4. The type of the secondary storage devices installed on this computer?
