Software Types

  1. Application software (or user application software) includes any applications that let users complete daily tasks. Application software depends on and cannot operate without the services of the system software.

Note, however, that it is sometimes hard to tell whether a piece of software is system software or application software, and there are big debates and even court case decisions on this question. Example: in the U.S. vs. Microsoft Corp. case, the debate is whether a browser is counted as system software or not.
