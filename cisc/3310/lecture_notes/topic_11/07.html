Software Types

To understand how operating systems relate to other programs, we should first learn what software types exist.

  1. System software is a collection of programs that sit on top of, control, and manage hardware. It provides services to other applications. System software includes:
    1. The operating system, and
    2. Utility software, which is not as essential as the OS but helps in analyzing the computer’s behavior, debugging errors, and optimizing its operation.
