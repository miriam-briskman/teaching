Boolean Symbols and Expressions

We write:

Logic GateBoolean Expression
NOT xx'
x AND yx$\cdot$y (or: xy)
x OR yx + y
x XOR yx ⊕ y
x NAND y(xy)'
x NOR y(x + y)'
x XNOR y(x ⊕ y)'

We can combine these expressions to create more complex logic expressions.
