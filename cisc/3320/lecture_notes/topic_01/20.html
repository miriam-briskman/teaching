Hardware

    1. RAM (Random-Access Memory) is the short-term memory of a computer. This memory has a limited size and is volatile (will be erased when the computer shuts down.)
    2. ROM (Read-Only Memory) stores a small program called BIOS that runs when the computer “boots” (turns on or restarts.) ROM is non-volatile, and cannot be deleted since it is "read-only".
    3. Cache is a temporary storage of data located between the CPU and RAM. Accessing cache is faster than accessing RAM in most cases.
