One-To-One Or Not: Horizontal Line Test

A graph can also tell us if a given function has a one-to-one property or not. This rule is called the Horizontal Line Test: if any horizontal line that you draw on top of the graph intersects at most once with the curve, then the function is injective; otherwise, if you can draw a horizontal line that intersects twice or more with the curve, then the function is NOT one-to-one.

Only one intersection with any horizontal line: injective ✔

Two or more intersections with a horizontal line: NOT injective ✕
