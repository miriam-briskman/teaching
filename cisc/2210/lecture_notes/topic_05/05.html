Functions

Fun Class Activities:

  1. Is the following relation considered a function: $R_1 = \{(0, 1), (1, -2), (0, 2), (-1, 1)\}$? Why or why not?
  2. Is the following relation considered a function: $R_2 = \{(2, 1), (3, 1), (1, 0), (4, 1)\}$? Why or why not?
  3. Let $f(x) = x + 3$. Find the following values of $f$:
    1. $f(2)$
    2. $f(0.5)$
    3. $f(-1)$
  4. Consider the function $g(x) = x^3$ with the domain $\mathbb{P}$.
    1. What is the range of $g$? List the first $4$ elements of the range.
    2. What is a convenient codomain for $g$?
    3. Describe $g$'s domain and codomain using the "$f: D \to C$" notation.
