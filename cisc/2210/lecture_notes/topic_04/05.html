Relations

Fun Class Activities:

  1. List all the elements (pairs) in the relation $\text{Equals}$ that is defined from $S = \{1, 2, 4, 15\}$ to $T = \{2, 5, 15, 21, 32\}$.
  2. List all the elements (pairs) in the relation $\text{EvenToOdd}$, which relates even integers to odd integers, that is defined from $S = \{1, 2, 4, 15\}$ to $T = \{2, 5, 15, 21, 32\}$.
  3. List all the elements (pairs) in the relation $\text{GreaterThan}$, or $>$, that is defined from $S = \{1, 2, 3, 4\}$ to $S = \{1, 2, 3, 4\}$.
