Set: Definition

To create the set of smilies $\{$😀$,$ 😁$,$ 😎$,$ 😜$\}$ in $\LaTeX$, first import the package 

\usepackage{twemojis}
and then use the following $2$ statements in the body of the document: 
\setlength​{\twemojiDefaultHeight}{0.5cm} % Increase the size of the emojies.

$\{\twemoji{grinning}, \twemoji{grin}, \twemoji{smiling face with sunglasses}, \twemoji{winking face with tongue}\}$

Tons of more smilies and emojis are listed in the following PDF documentation of the twemojis package: https://mirror.las.iastate.edu/tex-archive/macros/latex/contrib/twemojis/twemojis.pdf.
