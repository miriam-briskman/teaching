Information Security: Definitions

Example: Creating an email account, Joe Smith decided to set his account password to: '12345'.

Fun Class Activity: Match the items below to the concepts (one concept per item:)

Items

1Someone might guess the password and break into Joe's account.
2Programming the email server to accept only strong passwords.
3A password too short and too easy to guess.
4Mal Waring, a hacker, guesses Joe's password and breaks in.
5A program running password combinations against Joe's password to break it.

Concepts

Aa vulnerability
Ba threat
Can attack
Dan exploit
Ea control
