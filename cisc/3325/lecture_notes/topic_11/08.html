Virtual Private Networks: Types

VPN can be categorized as one of the following 2 types:

  1. Remote Access (Host-to-Site) VPN: Remote Access (Host-to-Site) VPN is a connection between a user and the LAN inside a company where the user is an employee who needs to connect to the corporate network from outside the company.

    This type of connection is used mainly by telecommuting or sales force employees who want to connect to the corporate LAN from remote locations.

    It provides a secure, encrypted communication between two parties that are connected via the Internet.

    Another variation of Remote Access connection includes Host-to-Host VPN, in which two host devices are connected through a VPN tunnel and transfer data with each other (but don't access any other resources on the network.)
